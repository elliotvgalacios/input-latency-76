# Input Latency 76

Input Latency 76 is a Python-based utility designed for gamers who want to measure and analyze the latency of their input devices. This tool offers precise insights into the responsiveness of your gaming peripherals, allowing you to optimize your setup for the best performance.

## Features

- **Real-Time Input Tracking**: Monitor input events in real time to determine the latency of your mouse, keyboard, or game controller.
- **Detailed Analysis**: Generate reports that showcase latency patterns, enabling you to identify performance bottlenecks caused by hardware or software configurations.
- **Customizable Settings**: Tailor measurement intervals and device profiles to suit your gaming style and preferences.
- **Cross-Platform Compatibility**: Works seamlessly on Windows, macOS, and Linux, making it accessible for gamers on any platform.

## Installation

To get started with Input Latency 76, clone the repository and install the necessary dependencies using pip:

```bash
git clone https://github.com/Developer/input-latency-76.git
cd input-latency-76
pip install -r requirements.txt
```

## Basic Usage Example

Once installed, you can run the tool from your terminal. Here’s a simple command to start measuring input latency:

```bash
python input_latency.py --device mouse
```

This command will initiate latency measurement for your mouse device. You can replace `mouse` with `keyboard` or `controller` to track other inputs. Check the output for detailed latency statistics.
